Detailed Description

template<typename VT, typename MT>
class blaze::TDVecTDMatMultExpr< VT, MT >

Expression object for transpose dense vector-transpose dense matrix multiplications.

The TDVecTDMatMultExpr class represents the compile time expression for multiplications between transpose dense vectors and column-major dense matrices.

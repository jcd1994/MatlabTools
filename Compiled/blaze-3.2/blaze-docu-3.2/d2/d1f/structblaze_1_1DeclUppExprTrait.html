Detailed Description

template<typename T>
struct blaze::DeclUppExprTrait< T >

Evaluation of the return type of a upper declaration (declupp) expression.

Via this type trait it is possible to evaluate the return type of a upper declaration (declupp) expression. Given the type T, which must be a matrix type, the nested type Type corresponds to the resulting return type. In case the type of T doesn't fit or if no declupp operation exists for the type, the resulting data type Type is set to INVALID_TYPE.

Detailed Description

template<typename T>
struct blaze::Has4Bytes< T >

Compile time size check.

This type trait offers the possibility to test whether a given type has a size of exactly four bytes. If the type T has four bytes, the value member constant is set to true, the nested type definition Type is TrueType, and the class derives from TrueType. Otherwise value is set to false, Type is FalseType, and the class derives from FalseType.

blaze::Has4Bytes<const int>::value // Evaluates to 'true' (on most architectures)
blaze::Has4Bytes<unsigned int>::Type // Results in TrueType (on most architectures)
blaze::Has4Bytes<volatile float> // Is derived from TrueType (on most architectures)
blaze::Has4Bytes<volatile double>::value // Evaluates to 'false'
blaze::Has4Bytes<const float>::Type // Results in FalseType
blaze::Has4Bytes<short> // Is derived from FalseType

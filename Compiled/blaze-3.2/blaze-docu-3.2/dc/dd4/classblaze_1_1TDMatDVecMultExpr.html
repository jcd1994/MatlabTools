Detailed Description

template<typename MT, typename VT>
class blaze::TDMatDVecMultExpr< MT, VT >

Expression object for transpose dense matrix-dense vector multiplications.

The TDMatDVecMultExpr class represents the compile time expression for multiplications between column-major dense matrices and dense vectors.

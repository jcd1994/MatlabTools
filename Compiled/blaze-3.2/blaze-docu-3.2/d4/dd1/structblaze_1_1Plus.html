Detailed Description

template<typename T1, typename T2>
struct blaze::Plus< T1, T2 >

Compile time integral addition.

The Plus alias declaration returns the sum of the two given template arguments T1 and T2. In order for Plus to be able to add the two types, both arguments are required to have a nested member value. The result of the addition can be accessed via the nested member value, the resulting type is available via the nested type ValueType.

blaze::Plus< Int<3> , Int<2> >::value // Results in 5
blaze::Plus< Long<3>, Int<2> >::ValueType // Results in long

Detailed Description

template<typename MT>
class blaze::NonNumericProxy< MT >

Access proxy for symmetric, square matrices with non-numeric element types.

The NonNumericProxy provides controlled access to the elements of a non-const symmetric matrix with non-numeric element type (e.g. vectors or matrices). It guarantees that a modification of element of the accessed matrix is also applied to element . The following example illustrates this by means of a sparse symmetric matrix with StaticVector elements:

using blaze::CompressedMatrix;
using blaze::StaticVector;
using blaze::SymmetricMatrix;
using Vector = StaticVector<int,3UL>;
// Creating a 3x3 symmetric sparses matrix
SymmetricMatrix< CompressedMatrix< Vector > > A( 3UL );
A(0,2) = Vector( -2, 1 ); // ( ( 0 0 ) ( 0 0 ) ( -2 1 ) )
A(1,1) = Vector( 3, 4 ); // => A = ( ( 0 0 ) ( 3 4 ) ( 5 -1 ) )
A(1,2) = Vector( 5, -1 ); // ( ( -2 1 ) ( 5 -1 ) ( 0 0 ) )

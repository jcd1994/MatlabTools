Detailed Description

template<typename MT, bool SO>
class blaze::SMatSerialExpr< MT, SO >

Expression object for the forced serial evaluation of sparse matrices.

The SMatSerialExpr class represents the compile time expression for the forced serial evaluation of a sparse matrix.

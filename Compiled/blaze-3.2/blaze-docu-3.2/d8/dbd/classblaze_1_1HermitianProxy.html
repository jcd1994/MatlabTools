Detailed Description

template<typename MT>
class blaze::HermitianProxy< MT >

Access proxy for Hermitian matrices.

The HermitianProxy provides controlled access to the elements of a non-const Hermitian matrix. It guarantees that a modification of element of the accessed matrix is also applied to element . The following example illustrates this by means of a dense Hermitian matrix:

// Creating a 3x3 Hermitian dense matrix
blaze::HermitianMatrix< blaze::DynamicMatrix<int> > A( 3UL );
A(0,2) = -2; // ( 0 0 -2 )
A(1,1) = 3; // => A = ( 0 3 5 )
A(1,2) = 5; // ( -2 5 0 )

BLAZE_USE_OPTIMIZED_KERNELS
#define BLAZE_USE_OPTIMIZED_KERNELS
Configuration of the streaming behavior.Via this compilation switch streaming (i.e. non-temporal stores) can be (de-)activated. For large vectors and matrices non-temporal stores can provide a significant performance advantage of about 20%. However, this advantage is only in effect in case the memory bandwidth of the target architecture is maxed out. If the target architecture&#39;s memory bandwidth cannot be exhausted the use of non-temporal stores can decrease performance instead of increasing it.

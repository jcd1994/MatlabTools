Detailed Description

template<typename VT, typename MT>
class blaze::TSVecSMatMultExpr< VT, MT >

Expression object for sparse vector-sparse matrix multiplications.

The TSVecSMatMultExpr class represents the compile time expression for multiplications between transpose sparse vectors and row-major sparse matrices.

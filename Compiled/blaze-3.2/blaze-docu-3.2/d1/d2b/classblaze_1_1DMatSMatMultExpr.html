Detailed Description

template<typename MT1, typename MT2, bool SF, bool HF, bool LF, bool UF>
class blaze::DMatSMatMultExpr< MT1, MT2, SF, HF, LF, UF >

Expression object for dense matrix-sparse matrix multiplications.

The DMatSMatMultExpr class represents the compile time expression for multiplications between a row-major dense matrix and a row-major sparse matrix.

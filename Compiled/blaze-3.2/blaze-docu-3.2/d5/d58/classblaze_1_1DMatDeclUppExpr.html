Detailed Description

template<typename MT, bool SO>
class blaze::DMatDeclUppExpr< MT, SO >

Expression object for the explicit upper declaration of dense matrices.

The DMatDeclUppExpr class represents the compile time expression for the explicit upper declaration of a dense matrix.

Compilation switch for the serial evaluation strategy of the upper declaration expression.

The useAssign compile time constant expression represents a compilation switch for the serial evaluation strategy of the upper declaration expression. In case the given dense matrix expression of type MT requires an intermediate evaluation, useAssign will be set to 1 and the upper declaration expression will be evaluated via the assign function family. Otherwise useAssign will be set to 0 and the expression will be evaluated via the subscript operator.

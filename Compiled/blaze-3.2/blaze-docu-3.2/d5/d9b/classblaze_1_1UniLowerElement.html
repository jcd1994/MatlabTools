Detailed Description

template<typename MT>
class blaze::UniLowerElement< MT >

Representation of an element within a sparse lower unitriangular matrix.

The UniLowerElement class represents an element (i.e. value/index pair) within a sparse lower unitriangular matrix. It guarantees that the unilower matrix invariant is not violated, i.e. that elements in the upper part of the matrix remain 0 and the diagonal elements remain 1. The following example illustrates this by means of a sparse lower unitriangular matrix:

using UniLower = blaze::UniLowerMatrix< blaze::CompressedMatrix<int> >;
// Creating a 3x3 lower unitriangular sparse matrix
UniLower A( 3UL );
A(1,0) = -2; // ( 1 0 0 )
A(2,0) = 3; // => A = ( -2 1 0 )
A(2,1) = 5; // ( 3 5 1 )
UniLower::Iterator it = A.begin( 1UL );
*it = 4; // Modification of matrix element (1,0)
++it;
*it = 9; // Invalid assignment to diagonal matrix element; results in an exception!

Detailed Description

template<typename MT, typename VT>
class blaze::SMatSVecMultExpr< MT, VT >

Expression object for sparse matrix-sparse vector multiplications.

The SMatSVecMultExpr class represents the compile time expression for multiplications between row-major sparse matrices and sparse vectors.
